Key Architecture Decisions

Multi-Tenant Strategy Schema-per-tenant in shared Aurora cluster for pilot phase, enabling cost efficiency while maintaining data isolation.
Authentication AWS Cognito User Pools with JWT tokens for stateless authentication and tenant identification.
Container Platform ECS Fargate for serverless container orchestration, eliminating infrastructure management overhead.
High Availability Multi-AZ deployment across Application Load Balancer, ECS tasks, and Aurora replicas.
